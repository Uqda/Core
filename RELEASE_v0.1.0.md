# Release v0.1.0 - January 16, 2026

## First Release of Uqda Core

This is the initial release of Uqda Core, a fork and rebranding of the Yggdrasil Network project.

## What's New

- **Complete Rebranding**: Rebranded from Yggdrasil Network to Uqda Network
- **New Module Path**: `github.com/Uqda/Core`
- **New Binary Names**: `uqda` and `uqdactl`
- **Updated Documentation**: All documentation updated to reflect Uqda branding

## Features

- End-to-end encrypted IPv6 mesh networking
- Self-organizing mesh topology
- Zero configuration required
- Support for Linux, Windows, and macOS
- Auto-configuration mode
- Static configuration mode
- Admin socket for monitoring
- IPv6 routing and addressing
- Multicast support
- TUN/TAP interface support

## Installation

### Pre-built Packages

Download pre-built packages from the [Releases](https://github.com/Uqda/Core/releases) page:

- **Linux**: `.deb` packages for Debian/Ubuntu
- **Windows**: `.msi` installers
- **macOS**: `.pkg` installers

### From Source

```bash
git clone https://github.com/Uqda/Core.git
cd Core
git checkout v0.1.0
./build
```

## Compatibility

- **Protocol Version**: 0.5 (fully compatible with Yggdrasil Network)
- **Go Version**: Requires Go 1.22 or later
- **Platforms**: Linux, Windows, macOS, FreeBSD, OpenBSD
