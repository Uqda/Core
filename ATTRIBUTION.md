### Original Project Description

Yggdrasil Network is an early-stage implementation of a fully end-to-end encrypted IPv6 network. It is lightweight, self-arranging, supported on multiple platforms, and allows pretty much any IPv6-capable application to communicate securely with other Yggdrasil nodes.

## This Fork

### Changes Made

This fork has been rebranded from Yggdrasil Network to Uqda Network with the following changes:

1. **Rebranding:** All references to "yggdrasil" have been changed to "uqda" throughout the codebase
2. **Module Path:** Changed from `github.com/yggdrasil-network/yggdrasil-go` to `github.com/Uqda/Core`
3. **Binary Names:** Changed from `yggdrasil` to `uqda` and `yggdrasilctl` to `uqdactl`
4. **Documentation:** Updated documentation with English content
5. **Repository:** Moved to https://github.com/Uqda/Core
6. **Rebranding Date:** January 16, 2026

### What Remains the Same

- **Core Functionality:** All networking protocols, encryption, and routing algorithms remain unchanged
- **License:** Maintains the same LGPL-3.0 license with binary distribution exception
- **Architecture:** No changes to the underlying network architecture or protocols
- **Compatibility:** Fully compatible with the original Yggdrasil Network protocol
