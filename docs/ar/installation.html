// التحميل

يمكن تحميل عُقدة للمنصات الرئيسية من صفحة GitHub Releases الخاصة بنا.

🚨 يرجى تذكر التحقق من تحديثات عُقدة بانتظام. لا تحتوي عُقدة على طريقة مدمجة لإشعارك عند توفر إصدارات جديدة. قد ترغب في الاشتراك في GitHub Releases Atom feed أو التفكير في مشاهدة مستودع GitHub.

// لينكس

عُقدة مدعومة جيداً على Linux. تتوفر حزم التوزيع:

ديبيان / أوبونتو
wget https://github.com/Uqda/Core/releases/latest/download/uqda-*.deb
sudo dpkg -i uqda-*.deb
من المصدر
git clone https://github.com/Uqda/Core.git
cd Core
./build
// ويندوز

عُقدة تعمل على Windows. قم بتنزيل مثبت MSI من GitHub Releases وقم بتشغيله.

أو قم بالبناء من المصدر باستخدام Go 1.25.8+.

// ماك أو إس

عُقدة مدعومة جيداً على macOS. قم بتنزيل مثبت .pkg من GitHub Releases أو قم بالبناء من المصدر.

// البدء السريع

بعد التثبيت، يمكنك بدء عُقدة في وضع الإعداد التلقائي:

sudo uqda -autoconf

أو قم بإنشاء ملف إعداد:

uqda -genconf > uqda.conf
sudo uqda -useconffile uqda.conf
