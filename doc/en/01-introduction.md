# Introduction to Uqda Network

## What is Uqda Network?

**Uqda Network** is an experimental decentralized mesh network protocol that provides a fully end-to-end encrypted IPv6 network. The project is an independent fork of the **Yggdrasil Network** project, with a complete rebranding while maintaining the same protocol and core technology.

## Core Concept

Uqda Network allows any connected device to communicate directly with any other device, without the need for central servers or traditional infrastructure. Every node acts as a router that can forward data to other nodes.

## Key Features

- 🔒 **End-to-End Encryption** - All data is fully encrypted
- 🌍 **Decentralized Network** - No single point of failure
- 🔄 **Self-Healing** - Network automatically repairs itself when links fail
- 🪶 **Lightweight** - Minimal resource consumption
- 🌐 **Works over IPv4/IPv6** - Works on any existing network
- 🎯 **Location Independent** - IPv6 address derived from cryptographic keys

## Difference from Yggdrasil

- ✅ **Technology:** Identical protocol
- ✅ **Identity:** New name, independent brand
- ✅ **License:** LGPL-3.0 (same license with attribution)
- ✅ **Development:** Independent and separate
- ❌ **Compatibility:** Separate networks (different addresses)

## Current Status

The project is in **Alpha** stage - the protocol works and is stable, but changes may occur in the future.
